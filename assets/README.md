# Logos y assets

Assets de La Calle de las Sirenas.

El logo fue extraido del PowerPoint `DOC-20260413-WA0128..pptx`, usando la version de mayor resolucion encontrada en `ppt/media/image1.jpg`.

Archivos:

- `logo-la-calle.webp`: version optimizada que usa la app.
- `logo-la-calle.png`: respaldo PNG optimizado.
- `logo-la-calle-source.jpg`: copia fuente extraida del PPTX.

Cuando tengas el logo oficial final, puedes reemplazar `logo-la-calle.webp` y `logo-la-calle.png`.

Ruta esperada por la app:

- `assets/logo-la-calle.webp`
- `assets/logo-la-calle.png`

Recomendacion:

- PNG transparente o WebP transparente.
- Minimo 1200 px de ancho si el logo es horizontal.
- Fondo transparente.
- Version en color claro si el logo se usa sobre fondos oscuros.

Si el archivo no existe, el reporte usa un placeholder tipografico.
